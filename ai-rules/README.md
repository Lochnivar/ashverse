# AI Rules Folder

This folder contains AI rules and guidelines that need to persist throughout the storyboard development process.

## Purpose

AI rules in this folder may include:
- Writing style guidelines
- Character voice rules
- Narrative tone requirements
- Thematic constraints
- World-building consistency rules
- Any other persistent guidelines for AI assistance

## Structure

AI rules should be organized by topic or type as needed.

## Usage

These rules should be referenced and followed when:
- Generating new content
- Editing existing content
- Analyzing story elements
- Making suggestions or recommendations
