# Ashverse - Epic Novel Series Storyboard

This repository contains the storyboard, planning materials, and world-building documentation for an epic series of novels.

## Structure

```
ashverse/
├── README.md                 # This file
├── series-overview.md        # High-level series overview
├── books/                    # Individual book planning
│   └── book-01-template.md
├── characters/               # Character profiles and arcs
│   └── character-template.md
├── world-building/          # Settings, lore, and world details
│   └── settings.md
├── plot/                    # Plot outlines and story arcs
│   └── plot-outline-template.md
├── scenes/                  # Individual scene breakdowns
│   └── scene-template.md
└── notes/                   # Research, ideas, and miscellaneous notes
    └── ideas.md
```

## Getting Started

1. Start with `series-overview.md` to define the overall vision
2. Create character profiles in `characters/`
3. Build your world in `world-building/`
4. Outline plots in `plot/`
5. Break down scenes in `scenes/`
6. Use `notes/` for research and random ideas

## Conventions

- Use markdown for all documentation
- Keep files focused and organized
- Link related documents using markdown links
- Version control your story development
