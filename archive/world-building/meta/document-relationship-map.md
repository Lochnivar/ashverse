## RELATIONSHIP TYPES

### Builds On
- Document A provides foundation, Document B expands on it
- Example: `point-of-divergence.md` → `gettysburg-master-timeline.md`

### References
- Document A mentions or links to Document B
- Example: `california-diplomacy.md` → `california-council-arbiter.md`

### Expands
- Document A is general, Document B provides detail
- Example: `native-super-states-master.md` → `native-super-states-embattled-survivors.md`

### Consolidates
- Document A combines information from multiple sources
- Example: `slavery-and-raids-master.md` (consolidates multiple topics)

### Parallel
- Documents cover similar topics from different angles
- Example: `wwi-timeline.md` and `wwi-without-us-entry.md`
