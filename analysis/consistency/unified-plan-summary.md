## KEY DECISIONS

### File Structure
- **Master Files:** 8 world-building master files (2000-5000 lines each)
- **Detail Files:** Keep only where valuable (> 500 lines, needed for editing)
- **Characters:** Consolidate to 3-5 family files
- **Analysis:** Archive most, keep active only

### Metadata Standard
- **Every file:** Date, Status, Purpose headers
- **Dependency tracking:** Hash-tracked dependencies section
- **Document hash:** Tracked at bottom

### Index Files
- **Master indexes:** Navigation and relationship maps
- **Quick reference:** Essential facts only
